# Getting Started

v0.1 is client-only. You bring your own database and implement the /metrics route on your Hono server.

## Install

```bash
bun add honolytics
```

## Setup

```tsx
import { HonolyticsProvider } from 'honolytics'

export default function App() {
  return (
    <HonolyticsProvider
      apiKey={process.env.NEXT_PUBLIC_ANALYTICS_KEY!}
      endpoint={process.env.NEXT_PUBLIC_ANALYTICS_URL!}
    >
      <YourApp />
    </HonolyticsProvider>
  )
}
```
