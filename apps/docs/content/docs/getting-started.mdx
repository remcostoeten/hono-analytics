## Prerequisites

- Node.js 18 or later
- pnpm 8 or later
- PostgreSQL 14+ or SQLite 3.35+

## Installation

### 1. Clone Repository

```bash
git clone https://github.com/your-org/onolythics.git
cd onolythics
pnpm install
```

### 2. Database Setup

Choose either PostgreSQL (recommended for production) or SQLite (development).

#### PostgreSQL

```bash
cd packages/backend
cp .env.example .env
```

Edit `.env`:

```bash
DATABASE_URL=postgresql://username:password@localhost:5432/analytics
DEFAULT_API_KEY=dev-key-12345
PORT=8000
```

The `DEFAULT_API_KEY` can be any string for development. For production, generate a secure random key:

```bash
openssl rand -hex 32
```

#### SQLite

```bash
DATABASE_URL=sqlite:./analytics.db
DEFAULT_API_KEY=dev-key-12345
PORT=8000
```

### 3. Run Migrations

```bash
cd packages/backend
pnpm db:generate
pnpm db:migrate
```

This creates:
- `projects` table for multi-tenancy
- `users` table for visitor tracking
- `sessions` table for session management
- `pageviews` table for page tracking
- `events` table for custom events

### 4. Start Backend

```bash
pnpm dev
```

Backend runs on `http://localhost:8000`.

### 5. Get Your API Key

Your API key authenticates requests to your analytics backend.

#### Quick Generation

Generate a secure API key instantly:

<ApiKeyGenerator label="Generate Production API Key" />

Copy the generated key to your `packages/backend/.env` file.

#### Or Use Simple Development Key

For quick testing, use this simple key in your `.env` file:

```bash
DEFAULT_API_KEY=dev-key-12345
```

For more details, see the [API Keys guide](/docs/api-keys).

## Tracking Setup

### React/Next.js

Install SDK:

```bash
pnpm add @onolythics/sdk
```

Wrap your app:

```tsx
import { AnalyticsProvider } from '@onolythics/sdk/react'

export function RootLayout({ children }) {
  return (
    <AnalyticsProvider
      apiKey="dev-key-12345"
      projectId="default-project"
      endpoint="http://localhost:8000"
    >
      {children}
    </AnalyticsProvider>
  )
}
```

Replace `dev-key-12345` with the API key from your `.env` file.

### Vanilla JavaScript

```js
import { initAnalytics, track } from '@onolythics/sdk'

const analytics = initAnalytics({
  apiKey: 'dev-key-12345',
  projectId: 'default-project',
  endpoint: 'http://localhost:8000'
})

track({
  url: window.location.href,
  durationMs: 5000
})
```

Replace `dev-key-12345` with the API key from your `.env` file.

## Dashboard Setup

Install hooks package:

```bash
pnpm add @onolythics/hooks
```

Create dashboard component:

```tsx
'use client'

import { useAnalytics } from '@onolythics/hooks'

export function Dashboard() {
  const { data, loading, error } = useAnalytics({
    config: {
      apiKey: 'dev-key-12345',
      endpoint: 'http://localhost:8000'
    },
    dateRange: {
      from: new Date(Date.now() - 7 * 24 * 60 * 60 * 1000),
      to: new Date()
    }
  })

  if (loading) return <div>Loading...</div>
  if (error) return <div>Error: {error}</div>
  if (!data) return null

  return (
    <div>
      <h1>Analytics</h1>
      <div>Total Users: {data.totals.users}</div>
      <div>Total Pageviews: {data.totals.pageviews}</div>
    </div>
  )
}
```

Replace `dev-key-12345` with the API key from your `.env` file.

## Verification

### Test Backend

```bash
curl http://localhost:8000/health
```

Expected response:

```json
{
  "status": "ok",
  "timestamp": "2024-01-01T00:00:00.000Z"
}
```

### Test Tracking

Visit your app and check the network tab for POST requests to `/track`.

### Test Metrics

```bash
curl -H "x-api-key: dev-key-12345" http://localhost:8000/metrics
```

Use the same API key from your `.env` file.
