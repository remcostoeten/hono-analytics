# API Reference

The HONO Analytics API is a REST API that allows you to track analytics events and retrieve metrics data.

## Base URL

```
http://localhost:8000  # Development
https://your-domain.com # Production
```

## Authentication

All API requests require authentication using an API key passed in the `x-api-key` header.

```bash
curl -H "x-api-key: your-api-key" \
     "http://localhost:8000/metrics"
```

## Headers

### Required Headers

| Header | Description | Example |
|--------|-------------|---------|
| `x-api-key` | Your project API key | `your-api-key` |
| `Content-Type` | Content type for POST requests | `application/json` |

### Optional Headers

| Header | Description | Example |
|--------|-------------|---------|
| `x-dev-traffic` | Mark traffic as development | `true` |

## Response Format

### Success Response

All successful responses follow this format:

```json
{
  "data": {},
  "timestamp": "2024-01-01T00:00:00.000Z"
}
```

### Error Response

Error responses include an error message and appropriate HTTP status code:

```json
{
  "error": "Invalid API key"
}
```

## Status Codes

| Code | Description |
|------|-------------|
| `200` | Success |
| `204` | Success (no content) |
| `400` | Bad Request |
| `401` | Unauthorized |
| `500` | Internal Server Error |

## Rate Limits

Currently, there are no rate limits implemented. This may change in future versions.
