# Getting Started

Welcome to your documentation site! This is built with Fumadocs, a powerful documentation framework.

## What's included

- **Fast**: Built on Next.js with App Router
- **Flexible**: Customizable themes and layouts
- **Full-text search**: Built-in search functionality
- **MDX Support**: Write content with React components

## Next Steps

Start editing the content in the `content/docs` directory to build your documentation.
