# HONO Analytics

Self-hosted web analytics focused on users, not events. Built with HONO and TypeScript.

## Why HONO Analytics?

- **Privacy-first**: Your data stays on your servers
- **User-centric**: Track users and sessions, not just events
- **Developer-friendly**: TypeScript SDK, React provider, vanilla JS support
- **Lightweight**: ~8KB gzipped, minimal performance impact
- **Session tracking**: 30-min inactivity timeout, automatic device detection

## Quick Install

```bash
# 1. Start backend
git clone [repo] && cd hono-analytics && pnpm install
docker-compose up -d postgres
./dev.sh

# 2. Install SDK
npm install @hono-analytics/sdk

# 3. Add to your app
import { AnalyticsProvider } from '@hono-analytics/sdk/react'
```
