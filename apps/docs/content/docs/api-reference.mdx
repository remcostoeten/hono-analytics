## Storage API

Storage adapters for persisting analytics data without external APIs.

- **[Local Storage](./api-reference/storage/local)** — Browser localStorage adapter ✅ **Production Ready**
- **[IndexedDB](./api-reference/storage/indexdb)** — Browser IndexedDB adapter ✅ **Production Ready**
- **[PostgreSQL](./api-reference/storage/postgres)** — PostgreSQL database adapter ⚠️ **Needs Testing**

## Quick Reference

### Basic Usage

```tsx
import { HonolyticsProvider, useAnalytics } from 'honolytics'

// HTTP Mode
<HonolyticsProvider apiKey="key" endpoint="/api/metrics">
  <App />
</HonolyticsProvider>

// Storage Mode
<HonolyticsProvider storageMode={true}>
  <App />
</HonolyticsProvider>

// In components
const { data, loading, error, refetch } = useAnalytics()
```

### Storage Setup

```tsx
import { initStorage, local, postgres } from 'honolytics/storage'

// Browser storage
await initStorage(local())

// Database storage
await initStorage(postgres({ url: 'postgresql://...' }))
```
